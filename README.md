# НЕЧТО — из глубокой бездны

> Локальный цифровой симулятор настольной игры в стиле Jackbox.  
> Один ноутбук на телевизоре — телефоны игроков как пульты.

---

## Как играть

1. Хост запускает приложение на ноутбуке/Steam Deck — экран выводится на ТВ
2. Игроки сканируют QR-код на экране со своих телефонов (Chrome / Firefox)
3. Раздача карт, ходы, обмены — всё с телефона

**4–12 игроков · Wi-Fi · без регистрации**

## Установка (Steam Deck)

```bash
tar -xzf nechto-linux-steamdeck.tar.gz -C ~/нечто
cd ~/нечто
bash install.sh
```

После `install.sh` ярлык появится в меню приложений Desktop Mode.

---

## Требования

- Хост: любой ноутбук / Steam Deck / Android TV с Wi-Fi
- Игроки: телефон с Chrome или Firefox (Safari не поддерживается)
- Общая Wi-Fi сеть
